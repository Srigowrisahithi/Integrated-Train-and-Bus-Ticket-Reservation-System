# Integrated Bus and Train Ticket Reservation System

## Description
This project is a web-based ticket reservation system that allows users to book bus and train tickets with real-time seat availability and booking management.

## Features
- Bus ticket booking
- Train ticket booking
- Seat availability checking
- PNR status tracking
- Ticket cancellation
- Booking history
- Payment simulation
- Admin panel for adding buses and trains

## Technologies Used
- PHP (Core PHP)
- MySQL (Database)
- HTML, CSS, JavaScript

## Modules
- Bus Reservation Module
- Train Reservation Module
- Admin Module

## How to Run
1. Install XAMPP / WAMP
2. Place project folder in `htdocs`
3. Import database in phpMyAdmin
4. Run `localhost/project-name`
