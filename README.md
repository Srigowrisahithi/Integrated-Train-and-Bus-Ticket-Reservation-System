# 🚌🚆 Integrated Bus and Train Ticket Reservation System

## 📌 Description
A web-based ticket reservation system that allows users to book bus and train tickets with real-time seat availability, booking management, and PNR tracking.

---

## ✨ Features
- 🚌 Bus ticket booking
- 🚆 Train ticket booking
- 🎫 Seat availability checking
- 🔍 PNR status tracking
- ❌ Ticket cancellation
- 📄 Booking history
- 💳 Payment simulation
- 🛠 Admin panel for adding buses and trains

---

## 🛠 Tech Stack
- **Backend:** PHP (Core PHP)
- **Database:** MySQL
- **Frontend:** HTML, CSS, JavaScript

---

## 📂 Modules
- Bus Reservation Module  
- Train Reservation Module  
- Admin Module  

---

1. Install **XAMPP / WAMP**
2. Place project folder in `htdocs`
3. Import database in **phpMyAdmin**
